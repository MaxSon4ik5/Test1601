Довгострокова мета: Backend NodeJS Developer

Этапи:

1. Вивчаємо IDE. Програма для роботи з кодом. Підсвітка коду
2. Вивчаємо git. Система версій. Github
3. Вивчаємо TypeScript + NodeJs
4. Вивчаємо Webpack
5. Вивчаємо модульність npm\yarn та принцип роботи
6. Вивчаємо Express Server
7. Вивчаємо Koa Server
8. Вивчаємо TypeORM
9. Піднімаємо проект на Express Server, який працює з TypeORM та БД PostgreSQL
10. Вивчаємо REST API та пробуємо витягувати дані за допомогою цієі технологіі та писати собі у Базу Даних
11. Підключаємо Linux  бібліотеку у себе на проекті та намагаємося використувувати іі функціонал як готове рішення, яким можна користуватися. Використувуємо Linux команди
12. Починаємо працювати над тестовим проектом для резюме
13. Плануємо проект, плануємо архітектуру та функціонал
14. Реалізуємо проект на тих технологіях що були вивчені
15. Налаштовуємо домен і викладуємо проект в інтернет
16. Створюємо резюме на англійській мові, вказуємо досвід, технологіі що освоіли та посилання на проект
17. Ходимо по співбесідам. Працюємо над слабкими сторонами, що будуть виявлені в процессі бесіди з розробниками. Вчимо додаткові технологіі та документацію по необхідності
18. Отримуємо першу роботу  Junior Backend NodeJS Developer і повністю полишаємо попередню для того щоб займатися розробкою кожен день, займатися професійно і бути актуальним спеціалістом на ринку праці. Одна робота - один проект. Концентруємося на проекті повністю для того щоб робити його якісно.
19. Пропрацювавши певний час на позиціі Junior коригуємо свое резюме, додаємо напрацьований досвід і влаштовуємося на роботу як Middle Developer з відповідною ЗП
20. Паралельно до цього всього записуємося на курси по англійській мові та прокачуємо саме розмовний рівень для того щоб можна було розглядати вакансіі з інших краін і вільно спілкуватися з англомовними замовниками по іхнім проектам.
